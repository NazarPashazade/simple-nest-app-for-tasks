## Setup

### Prerequisites

- Node.js (v14 or above)
- npm or yarn


### Installation

1. Clone the repository:

```
git clone <repository-url>
```

## HOW TO TEST

```
GET: http://localhost:3000/api/people/employed
```
 

  
## Commands


- **Run Locally**  

```
npm run start:local
```

- **Commands to Test**  

```
npm run test

npm run test:watch

npm run test:cov
```
